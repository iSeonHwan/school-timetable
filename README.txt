=====================================
  학교 시간표 관리 시스템 — 실행 방법
=====================================

[ 사전 설치 ]
터미널에서 아래 명령 실행:

  pip install PyQt6 SQLAlchemy psycopg2-binary

  (PostgreSQL 미사용 시 psycopg2-binary 생략 가능)

[ 실행 ]
  python3 main.py

[ 데이터베이스 설정 ]
- 기본값: SQLite (같은 폴더에 timetable.db 파일 자동 생성)
- 네트워크 공유 원할 경우:
  앱 실행 후 좌측 하단 "⚙ DB 연결 설정" → PostgreSQL 선택 →
  서버 IP, 포트, DB명, 계정 입력 → 확인

[ 사용 순서 ]
  1. 편제 설정 → 학년/반 입력
  2. 교사 관리 → 교사 추가 + 불가 시간 설정
  3. 교과목/시수 → 교과목 추가 + 반별 시수 배정
  4. 학기 추가 → 좌측 "+ 학기 추가"
  5. 자동 생성 → "▶ 자동 생성" 클릭
  6. 시간표 조회 → 반별 / 교사별 탭 조회

[ 구현된 기능 ]
  ✅ 편제 입력 (학년/반)
  ✅ 교사 정보 + 불가 시간 설정
  ✅ 교과목 + 주당 시수 배정
  ✅ 교실/특별실 관리
  ✅ 제약조건 기반 시간표 자동 생성
  ✅ 시간표 조회 — Mode A (요일×교시, 학반별)
  ✅ 시간표 조회 — Mode B (교시×학반, 1일 전체)
  ✅ 교사별 시간표 조회
  ✅ SQLite / PostgreSQL 선택 가능

[ 다음 단계 예정 기능 ]
  - 당일 시간표 수정 + 변경 신청/결재
  - 학사일정 관리
  - PDF 출력
  - 변경 이력 관리
  - NEIS 내보내기
